Welcome to Viewport

A simplified way to test responsive website layouts at common resolutions.

Recently I've been doing some responsive design work and I found that checking screen resolutions was a bit of a pain. Dragging the window size works adequately for watching responsive styles "step-down", but it's tricky so get it pixel perfect and my browser windows won't go smaller than 400px wide.

I couldn't find any free simple tools to do exactly what I wanted (including hiding the scrollbar), so I decided to build my own. This is what I made - I found it to be quite useful and I hope you will too.

Browser specific features: WebKit browsers allow you to hide the scrollbar if you wish (Chrome and Safari). I recommend you use Chrome because Safari's iframes tend to misbehave for me. I don't recommend using this on a mobile device, but feel free to try it.
